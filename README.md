# Student Activity Manager

## 📌 Project Description
This is a React-based project that manages student activities and attendance.
It built using React functional components.
It demonstrates useState, useRef, conditional rendering, props, and event handling.
No backend or localStorage is used, so data resets on refresh as required.

## 🧱 Project Architecture
/src/components/
- StudentForm.jsx
- StudentList.jsx
- StudentCard.jsx

App.jsx – Main module rendering

## ⚙️ Features
- Add Student
- Delete Student
- Toggle Details
- Mark Attendance
- Ref-based input focus
- Conditional Rendering

## 🚫 Restrictions Followed

- No Backend Used
- No Database
- No localStorage
- Data resets on refresh
- 
## 🧠 Modules Explanation

### StudentForm.jsx
Handles input using useRef and submits student data.

### StudentList.jsx
Displays list using map() and conditional rendering.

### StudentCard.jsx
Handles delete, toggle details, and attendance marking.

### App.jsx
Main state management using useState.
